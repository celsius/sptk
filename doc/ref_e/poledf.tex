\name{poledf}{all pole digital filter for speech synthesis}%
{digital filter}

\begin{synopsis}
\item[poledf] [ --m $M$ ] [ --p $P$ ] [ --i $I$ ] [ --t ] [ --k ]
              {\em afile} [ {\em infile} ]
\end{synopsis}

\begin{qsection}{DESCRIPTION}
入力データを {\em afile}の線形予測係数 $K,a(1),\ldots,a(M)$ をもつ
全極標準形フィルタによりフィルタリングし，標準出力に出力します．
\par
Input and output data are in float format.
\par
全極標準形フィルタの伝達関数 $H(z)$ は，
\begin{displaymath}
H(z) = \frac{K}{\displaystyle 1+\sum_{m=1}^M a(m) z^{-m}}
\end{displaymath}
です．
\end{qsection}

\begin{options}
	\argm{m}{M}{order of coefficients}{25}
	\argm{p}{P}{frame period}{100}
	\argm{i}{I}{interpolation period}{1}
	\argm{t}{}{transpose filter}{FALSE}
	\argm{k}{}{filtering without gain}{FALSE}
\end{options}

\begin{qsection}{EXAMPLE}
float形式のピッチデータ {\em data.pitch} から励振源を作成し，
線形予測係数ファイル {\em data.lpc} により標準形合成フィルタを駆動し，
合成音声を {\em data.syn} に出力する:
\begin{quote}
  \verb!excite < data.pitch | poledf data.lpc > data.syn!
\end{quote}
\end{qsection}
