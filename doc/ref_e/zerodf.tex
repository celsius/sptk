\name{zerodf}{all zero digital filter for speech synthesis}{digital filter}

\begin{synopsis}
\item[zerodf] [ --m $M$ ] [ --p $P$ ] [ --i $I$ ] [ --t ] [ --k ]
		{\em bfile} [ {\em infile} ]
\end{synopsis}

\begin{qsection}{DESCRIPTION}
指定されたファイルから読み込まれたデータを {\em bfile} の係数，
$b(0), b(1), \ldots, b(M)$をもつ全零標準形フィルタによりフィルタリングし，
標準出力に出力します．
\par
Input and output data are in float format.
\par
全零標準形フィルタの伝達関数 $H(z)$は，
\begin{displaymath}
H(z) = \sum_{m=0}^{M} b(m) z^{-m}
\end{displaymath}
と表されます．
\end{qsection}

\begin{options}
	\argm{m}{M}{order of coefficients}{25}
	\argm{p}{P}{frame period}{100}
	\argm{i}{I}{interpolation period}{1}
	\argm{t}{}{transpose filter}{FALSE}
	\argm{k}{}{filtering without gain}{FALSE}
\end{options}

\begin{qsection}{EXAMPLE}
float形式のピッチデータ {\em data.pitch} から励振源を作成し，零型フィルタの
係数ファイル {\em data.b} により全零フィルタを駆動し，合成音声を
 {\em data.syn}に出力する:
\begin{quote}
  \verb!excite < data.pitch  | zerodf data.b > data.syn!
\end{quote}
\end{qsection}
