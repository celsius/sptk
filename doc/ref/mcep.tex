\name[ref:mcep-IEICE,ref:amcep-ICASSP92]{mcep}{メルケプストラム分析}%
{音声分析}

\begin{synopsis}
\item [mcep] [ --a $A$ ] [ --m $M$ ] [ --l $L$ ] [ --i $I$ ] [ --j $J$ ] 
	     [ --d $D$ ] [ --e $E$ ] [ {\em infile} ]
\end{synopsis}

\begin{qsection}{DESCRIPTION}
メルケプストラム分析を行い，
メルケプストラム係数 $c_{\alpha}(m)$ を標準出力に出力します．
入力は長さ$L$ の時系列
\begin{displaymath}
  x(0),x(1),\ldots,x(L-1)
\end{displaymath}
です．
\par
データ形式は入力，出力ともfloat 形式です．
\par
メルケプストラム分析では，音声のスペクトルを $M$ 次の
メルケプストラム $c_{\alpha}(m)$ により
\begin{displaymath}
H(z) = \exp \sum_{m=0}^M c_{\alpha}(m) \tilde{z}^{-m} 
\end{displaymath}
とモデル化し，対数スペクトルの不偏推定法
における評価関数を適用します．
ここで，$\tilde{z}^{-1}$ は1次のオールパス関数
\begin{displaymath}
\tilde{z}^{-1} = \frac{z^{-1}-\alpha}{1-\alpha z^{-1}}
\end{displaymath}
であり，その位相特性は標本化周波数10kHzの場合 $\alpha=0.35$，
標本化周波数8kHzの場合 $\alpha=0.31$ と選べば，
人間の音の高さに対する聴覚特性を表すメル尺度をよく近似します．
\par
ここでは，評価関数を最小にするメルケプストラムを求めるために，
Newton--Raphson法が用いられています．
\end{qsection}

\newpage
\begin{options}
	\argm{a}{A}{周波数圧縮パラメータ $\alpha$．}{0.35}
	\argm{m}{M}{分析次数．}{25}
	\argm{l}{L}{フレーム長．}{256}
	\desc[1zh]{通常，以下のオプションの指定は必要ありません．}
	\argm{i}{I}{Newton-Raphson 法の最小反復回数．}{2}
	\argm{j}{J}{Newton-Raphson 法の最大反復回数．}{30}
	\argm{d}{D}{Newton-Raphson 法の終了条件．\\
		    $\varepsilon^{(i)}$ の繰り返しによる変化率が
		    $D$ 以内になったときに終了．}{0.001}
	\argm{e}{E}{ピリオドグラムに足し込む小さな値．}{0.0}
\end{options}

\begin{qsection}{EXAMPLE}
float 形式の音声データ {\em data.f} を分析し，{\em data.mcep} に
メルケプストラム係数を得る:
\begin{quote}
 \verb!frame < data.f | window | mcep > data.mcep !
\end{quote}
\end{qsection}
