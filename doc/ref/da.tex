\name{da}{オーディオデータの再生}{DA変換}
\begin{synopsis}
\item [da] [ --s $S$ ] [ --c $C$ ] [ --g $G$ ] [ --a $A$ ] [ --o $O$ ] 
	   [ --w ] [ --H $H$ ]
\item [\ ~~~] [ --v ] [ +$type$ ] [ {\em infile1} ] [ {\em infile2} ] ...
\end{synopsis}

\begin{qsection}{DESCRIPTION}
 {\em da}は，指定されたファイルのデータをスピーカーまたはヘッドフォンか
 ら再生します．ファイルが指定されなかった場合，データは標準入力から読み
 込まれます．
 オーディオデバイスの対応していない標本化周波数の場合，適切な周波数にア
 ップサンプリングして再生します．
 
 16bit オーディオデバイスの使用可能な Linux (i386) と FreeBSD 
 (i386 newpcm driver)，SunOS 4.1.x，SunOS 5.x (SPARC) に対応しています．
 
 オプションは環境変数で設定することもできます．

\begin{tabular}{ll}
%% DA\_SAMPFREQ & 標本化周波数\\
DA\_GAIN & ゲイン\\
DA\_AMPGAIN & 振幅ゲイン\\
DA\_PORT & 出力ポート\\
DA\_HDRSIZE &ヘッダサイズ\\
DA\_FLOAT & 入力データ型をfloat型に指定\\
\end{tabular}

\end{qsection}

\begin{options}
	\argm{s}{S}{標本化周波数 (kHz)．指定可能な値は 8，10，11.025，
		12, 16，22.05，32，44.1，48
		(但し，11.025，22.05，44.1はそれぞれ 11，22，44 に省略可能) です．}{16}
	\argm{g}{G}{ゲイン．入力データを $2^G$ 倍して出力します．}{0}
	\argm{a}{A}{オーディオデバイスの振幅ゲイン(0..100)．
		指定しない場合はそれまでの設定が適用されます．}{N/A}
	\argm{o}{O}{出力ポート(s : スピーカ，h : ヘッドフォン)．SPARC版のみ有効．}{s}
	\argm{w}{}{入力データのバイトスワップを行う．}{FALSE}
	\argm{H}{H}{ヘッダサイズ(バイト数)．}{0}
	\argm{v}{}{データファイル名を表示．}{FALSE}
	\argp{type}{入力データ型(s : short型，f : float型)．}{s}
\end{options}

\begin{qsection}{EXAMPLE}
標本化周波数8kHzのshort型の音声データファイル {\em data.s} をヘッドフォンから
再生する:
\begin{quote}
\verb! da -s 8 -o h data.s!
\end{quote}
\end{qsection}

\begin{qsection}{BUGS}
linux版では，出力ポートの指定ができません．
\end{qsection}
