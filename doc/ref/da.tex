\name{da}{オーディオデータの再生}{DA変換}
\begin{synopsis}
\item [da] [ --s $S$ ] [ --c $C$ ] [ --g $G$ ] [ --a $A$ ] [ --o $O$ ] 
	   [ --w ] [ --H $H$ ]
\item [\ ~~~] [ --v ] [ +$type$ ] [ {\em infile1} ] [ {\em infile2} ] ...
\end{synopsis}

\begin{qsection}{DESCRIPTION}
 {\em da}は，指定されたファイルのデータをスピーカーまたはヘッドフォ
 ンから再生します．ファイルが指定されなかった場合，データは標準入力から
 読み込まれます．
 実体はシェルスクリプトで，内部でサンプリングレート変換にusを，オーディ
 オデバイスの制御にdawriteを使用します．{\bf --c，--g，--a，--o，--w，--H}
 オプションはdawriteに渡され，またdawriteと同様に環境変数で設定することもできます．

\begin{tabular}{ll}
%% DA\_SAMPFREQ & 標本化周波数\\
DA\_GAIN & ゲイン\\
DA\_AMPGAIN & 振幅ゲイン\\
DA\_PORT & 出力ポート指定\\
DA\_HDRSIZE &ヘッダサイズ\\
DA\_FLOAT & 入力データ型をfloat型にする\\
\end{tabular}

\end{qsection}

\begin{options}
	\argm{s}{S}{標本化周波数．（ 8，10，11.025，12, 16，22.05，32，
 44.1，48 kHz）但し，11.025，22.05，44.1はそれぞれ11，22，44に省略可能}{16}
	\argm{g}{G}{ゲイン．（..，-2，-1，0，-1，2，..）}{0}
	\argm{a}{A}{振幅ゲイン．（0..100）指定しない場合はそれまでの設定が適用されます．}{N/A}
	\argm{o}{O}{出力ポート指定．（s : スピーカ，h : ヘッドフォン．SPARC版のみ有効）}{s}
	\argm{w}{}{入力データのバイトスワップを行う}{FALSE}
	\argm{H}{H}{ヘッダサイズ（バイト単位）．}{0}
	\argm{v}{}{データファイル名を表示．}{FALSE}
	\argp{type}{入力データ型．（s : short型，f : float型）}{s}
\end{options}

\begin{qsection}{EXAMPLE}
標本化周波数8kHzのshort型の音声データファイル {\em data.s} をヘッドフォンから
再生する:
\begin{quote}
\verb! da -s 8 -g -1 -o h data.s!
\end{quote}
\end{qsection}

\begin{qsection}{BUGS}
linux版では，出力ポートの指定ができません．
\end{qsection}
