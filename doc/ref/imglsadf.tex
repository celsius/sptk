\name[ref:MGLSA-IECE]{imglsadf}{音声合成のための逆MGLSAフィルタ}
{音声合成用フィルタ}

\begin{synopsis}
\item [imglsadf] [ --m $M$ ] [ --a $A$ ] [ --g $G$ ] [ --p $P$ ]
		 [ --i $I$ ]  [ --t ]  [ --k ]
\item [\ ~~~~~~~~~] {\em mgcfile} [ {\em infile} ]
\end{synopsis}

\begin{qsection}{DESCRIPTION}
{\em mgcfile}のメル一般化ケプストラム係数 $c_{\alpha,\gamma}(m)$
で与えられる伝達関数の逆関数をもつ MGLSA フィルタにより，入力データを
フィルタリングして標準出力に出力します．
\par
データ形式は入力，出力ともfloat 形式です．
\end{qsection}

\begin{options}
	\argm{m}{M}{メル一般化ケプストラムの次数．}{25}
	\argm{a}{A}{周波数圧縮パラメータ $\alpha$．}{0.35}
	\argm{g}{G}{一般化ケプストラムのべきパラメータ $\gamma$．\\
			 ただし，$G>1.0$ のときは $\gamma=-1/G$．}{1}
	\argm{p}{P}{係数の更新周期．}{100}
	\argm{i}{I}{係数の補間周期．}{1}
	\argm{t}{}{転置型フィルタ．}{FALSE}
	\argm{k}{}{ゲインを除いたシステム関数でフィルタリングする}{FALSE}
\end{options}

\begin{qsection}{EXAMPLE}
float形式の音声データファイル {\em data.f} をメル一般化ケプストラム分析
(15次，$\alpha=0.35$，$\gamma=-1/2$)
の結果に基づいて，逆フィルタを出力し，{\em data.e}を求める:
\begin{quote}
 \verb!frame < data.f | window | \!\\
 \verb!    mgcep -m 15 -a 0.35 -g -0.5 > data.mgc!\\
 \verb!imglsadf -m 15 -a 0.35 -g -0.5 data.mgc < data.f \!\\
 \verb!    > data.e!
\end{quote} 
\end{qsection}

\begin{qsection}{BUGS}
$n$ を自然数として，$\gamma = -1/n$ の場合にしか対応していない．
\end{qsection}
