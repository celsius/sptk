\name{glogsp}{対数スペクトルのプロット}{none}

\begin{synopsis}
\item[glogsp] [ --O $O$ ] [ --x $X$ ] [ --y $ymin \; ymax$ ] [ --ys $YS$ ] 
              [ --p $P$ ] [ --ln $LN$ ] 
\item[\ ~~~~~~~] [ --s $S$ ] [ --l $L$ ] [ --c $comment$ ] [ {\em infile} ]
\end{synopsis}

\begin{qsection}{DESCRIPTION}
{\em glogsp}は，float型のデータを標準入力から読み込んで，それをグラフ表示
するプロッタのコマンド列を生成します．
従って，{\em xgr}コマンド等と併用して使います．
\par
実体はシェルスクリプトで，内部では{\em fig}と{\em fdrw}を用いています．
\end{qsection}

	\argm{help}{}{詳細なヘルプを表示．}{}
\end{options}

\begin{qsection}{EXAMPLE}
short形式の音声データ (標本化周波数10kHz) のファイル {\em data.s} の
対数振幅スペクトルを求め，そのグラフを画面に描画する:
\begin{quote}
 \verb!x2x +sf data.s | bcut -s 4000 -e 4255 | window -n 2| spec |\! \\
 \verb!glogsp -x 5 | xgr!
\end{quote}
